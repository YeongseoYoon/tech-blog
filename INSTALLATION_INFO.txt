📍 프로젝트 위치: ~/tech-blog

🚀 빠른 시작:
   cd ~/tech-blog
   npm run dev
   # http://localhost:3000 열기

📋 프로젝트 정보:
   • Next.js 15 + React 19 + TypeScript 5
   • Tailwind CSS 3 + Remark
   • 완전한 기술 블로그 시스템

✨ 구현된 기능:
   ✅ .md 파일로 글 작성
   ✅ 슬러그 기반 라우팅 (/blog/slug)
   ✅ 동적 목차 (H1-H4 자동 추출)
   ✅ 텍스트 검색 (제목/요약/내용)
   ✅ 해시태그 필터링
   ✅ Giscus 댓글 기능
   ✅ 반응형 디자인

📁 주요 파일:
   • app/page.tsx             - 홈/글 목록 페이지
   • app/blog/[slug]/page.tsx - 글 상세 페이지
   • components/             - 재사용 컴포넌트
   • lib/mdParser.ts         - 마크다운 파싱
   • posts/                  - 마크다운 글 저장소

📚 문서:
   • README.md               - 전체 가이드
   • GETTING_STARTED.md      - 시작 가이드
   • PROJECT_SUMMARY.md      - 프로젝트 요약

🎯 다음 단계:
   1. npm run dev 로 개발 서버 시작
   2. posts/에 새 .md 파일 추가
   3. 브라우저에서 확인
   4. Giscus 설정 (선택사항)
   5. 배포 (Vercel 추천)

💡 마크다운 포스트 작성 예:
   ---
   title: '제목'
   date: '2025-03-10T10:00:00.000Z'
   summary: '요약'
   tags: ['Tag1', 'Tag2']
   ---
